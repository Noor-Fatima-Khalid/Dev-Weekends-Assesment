# Dev-Weekends-Assesment
# Pomodoro Timer

A minimal, polished Pomodoro focus timer built with React + Vite.
## Luive Deplyment URL:
https://dev-weekends-assesment.vercel.app/

## Run locally

**Requirements:** Node.js 18+

```bash
npm install
npm run dev
```

Then open http://localhost:5173

**Build for production:**

```bash
npm run build
npm run preview
```

## Features

- 25/5 focus/break cycles (fully configurable)
- Circular progress ring with smooth animation
- Audible bell cue via Web Audio API (no audio files needed)
- Daily session history — persists across reloads, resets at midnight
- Background and ring color shift between focus (purple) and break (teal) modes
- Responsive down to 360px
- Keyboard accessible, ARIA labelled

## Stack
React 18+ · React Router v6 · Vite · CSS Modules · Web Audio API · localStorage
